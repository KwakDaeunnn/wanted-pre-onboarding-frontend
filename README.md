## 프로젝트의 실행 방법
1. repository를 git clone 합니다.<br/>
   `git clone https://github.com/KwakDaeunnn/wanted-pre-onboarding-frontend.git` <br/>
3. 패키지를 설치합니다. <br/>
   `npm install` <br/>
4. 리액트를 실행합니다. <br/>
   `npm start` <br/>
